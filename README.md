# 柳州模拟联合国大会 / 柳州高中模联协会 官网

本网站/项目由渊梦文创为柳州高中模拟联合国协会制作，为柳高模联所办会议提供便捷的报名方式，为相关人员提供可靠的信息获取渠道。

本网站部署于Github Pages，通过又拍云CDN进行访问加速。
